- **代码组织**: 复用基类，拆分复杂逻辑为小函数，使用类型注解。
- **请求构造**: `meta` 传递上下文，合理 `retry_times/timeout`，优先 `curl` 下载器。
- **Step 存储规范**: 只用 `StorageManager` 进行文件操作，控制单目录文件数，保持字段稳定。
- **导出器**: 继承 `BaseExporter`，统计数据、生成 README 与映射文件。
- **可观测性**: 埋点关键指标，错误要有分类与上下文。
